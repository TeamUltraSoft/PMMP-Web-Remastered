What's PMMP

PMMP or PocketMine-MP is a custom server software for the Minecraft: Bedrock family of Minecraft editions (includes Android, iOS, W10 and others) that is written in PHP.

Pocketmine-MP is the ultimate Minecraft: Bedrock Edition Server Software with limitless customization, full gameplay control and is 100% open source and free.
