What's PMMP

PMMP or PocketMine-MP is a custom server software for the Minecraft: Bedrock family of Minecraft editions (includes Android, iOS, W10 and others) that is written in PHP.

Pocketmine-MP is the ultimate Minecraft: Bedrock Edition Server Software with limitless customization, full gameplay control and is 100% open source and free.

Features

Plugin Support

Extend the game in the way you want, add awesome features.

Multi-world System

Load different levels at once, and teleport back and forth.

Unlimited Control

Teleport players, whitelist your server, tune the server, Remote Console

Unlimited Players

Get all your friends in one server. Or run a public server.

Cheat Protection

Disables flying, item hack, running & more. With an On/Off switch.

Even more!

Endless features, and we continuously implement new things.
